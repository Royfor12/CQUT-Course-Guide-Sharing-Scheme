### 网页端上传

1. 登录 GitHub 网页端。如果没有 GitHub 账号，你需要花几分钟注册一下。
2. 点击 `Fork`【复制】本项目。
3. 打开对应文件夹，或新建文件夹（直接修改浏览器地址栏中的路径并回车，GitHub 会自动创建新文件夹）。
4. 点击 `Upload`，上传你的文件。
5. 对上传的文件进行核查，确认没有错误提交如下内容：
     - 重复的文件
     - 冗余二进制文件
     - 侵犯他人版权的文件
     - 与当前科目无关的文件
     - 其它路径下已存在的文件
6. 核查无误后，在自己的项目中点击上方的 `Pull requests` 向本仓库提交合并请求。

## ⚡ 前言

网络冲浪🏄时发现了浙江大学的课程攻略共享计划与上海交通大学生存手册，深受震撼。

故此也想建立一个重庆理工大学课程攻略计划，内容包括各类课程复习资料。

我们同在重庆理工大学，以相同的身份，学习相同的课程，师从相同的教授。教授们往往为了省事，几年的课程资料、期末试题通常变化不大。学习这些资料，阅读学长学姐留下的复习攻略，可能会减轻信息不对称对你造成的影响，让你在复习时更有针对性。面对年复一年、无数学子独立摸索、重复劳动的现状，我不禁感到惋惜与不平。

再加之本人有数字仓鼠🐹症。

因此，我希望建立一个项目，把那些零散、口耳相传、难以获取的知识与资源整理出来，让它们更公开、更易获取，并能由大家共同维护、持续完善。本项目非营利、永久免费开源，旨在推动知识传播、提升资源质量、降低获取成本、倡导无纸化学习、促进教育公平，让西唯兵学子在备考、复习之时不必再四处苦寻资料。

借用浙大的课程攻略中的一句话：**我希望只要是前人走过的弯路，后人就不必再走。这是我的信念，也是我建立这个项目的原因。**
